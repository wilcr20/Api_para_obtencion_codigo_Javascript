Consideraciones generales

  1. En todo momento requerirá de una conexión a internet. Esto porque la librería aún no posee un descargable que permita utilizar las funciones de manera local.
  2. Debe registrarse en el sistema e iniciar sesión para tener acceso a todas las funciones registradas. Además, usted podrá registrar nuevas funciones.
  3. Existe una sección del sistema que le permite visualizar las funciones registradas y filtrar los resultados de acuerdo a sus necesidades. El mismo sistema le facilita la obtención del link de importación para cada una de las funciones
  4. Si la función que requiere importar depende de otras funciones registradas en el sistema, usted no debe importar dichas funciones individualmente. El sistema carga automáticamente las funciones de las que depende la función a importar.
  5. No debe preocuparse si comete un error e importa dos veces la misma función utilizando el núcleo de la librería. Ya que el mismo sistema se encarga de verificar esto y realiza las acciones necesarias para evitar que su página web se vea afectada.

Registro de funciones

Para registrar una función en la librería de javascript dinámica se debe especificar la siguiente información:

  1. Nombre de la función: En JDL aspiramos a la excelencia. Por lo que solicitamos a las personas que utilicen el sistema que utilicen nombres representativos para sus funciones. Ya que esto facilita la búsqueda de la misma.

  2. Descripción de la función: Breve y concisa. Que considere todos los aspectos de relevancia con respecto al cómo opera la función.

  3. Código javascript: Código fuente de la función. Utilizar nombres de variables representativos.

  4. Etiquetas: Se trata de una categoría para la función. Seleccionar la o las categorías indicadas para su función provocará que está sea fácil de encontrar. El sistema posibilita que indique etiquetas tan solo con seleccionarlas. Si la etiqueta que desea no se encuentra en la lista de etiquetas puede optar por ingresar una, siguiendo siempre el estándar de utilizar la primera letra en mayúscula y el resto en minúsculas.

  5. Dependencias: En caso de que su función necesite utilizar otra función conocida por el sistema para operar deberá indicarlo a través de la interfaz. Solo debe seleccionar aquellas funciones de las que depende su función, de esta manera el sistema podrá cargar todas las dependencias cuando su función sea importada en cualquier página web.

Edición de funciones

Existe la posibilidad de modificar cada una de las funciones del sistema. No obstante, deben considerarse los siguientes aspectos:

  • Al editar una función, el sistema le cargará automáticamente toda la información de la misma. Esto incluye el nombre, descripción, código javascript, las etiquetas y dependencias. Así, queda en manos del usuario el eliminar o agregar etiquetas y/o dependencias y la manipulación de los demás datos de la función.

  • Cada edición de la función implica la generación de una nueva. Al editar una función se crea una versión de la misma que se diferencia por los cambios realizados. De esta manera la cantidad de versiones que puede tener una función corresponde a la cantidad de veces que haya sido editada.







Búsqueda de funciones

En el caso de la búsqueda de funciones, se podrá realizar un filtrado muy completo sobre todas las funciones existentes en el sistema. Es posible conocer el código javascript, la url de importación y las versiones para cada una de las funciones registradas.

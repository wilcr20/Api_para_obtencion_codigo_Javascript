En esta sección se abordan los siguientes aspectos:

  • Consideraciones Generales
  • Importación del núcleo de la librería
  • Importación individual de funciones

Consideraciones generales

  1. En todo momento requerirá de una conexión a internet. Esto porque la librería aún no posee un descargable que permita utilizar las funciones de manera local.
  2. Debe registrarse en el sistema e iniciar sesión para tener acceso a todas las funciones registradas. Además, usted podrá registrar nuevas funciones.
  3. Existe una sección del sistema que le permite visualizar las funciones registradas y filtrar los resultados de acuerdo a sus necesidades. El mismo sistema le facilita la obtención del link de importación para cada una de las funciones
  4. Si la función que requiere importar depende de otras funciones registradas en el sistema, usted no debe importar dichas funciones individualmente. El sistema carga automáticamente las funciones de las que depende la función a importar.
  5. No debe preocuparse si comete un error e importa dos veces la misma función utilizando el núcleo de la librería. Ya que el mismo sistema se encarga de verificar esto y realiza las acciones necesarias para evitar que su página web se vea afectada.

El núcleo de la librería consta básicamente de una función que permite cargar las funciones deseadas. Dicha función tiene como nombre importarDesdeJDL . A continuación se presenta un ejemplo de uso:



En la imagen anterior se puede visualizar la importación del núcleo de la librería. Y en el ejemplo se está importando la función con el identificador 45.

Básicamente se requiere una etiqueta script cuyo valor en el atributo src sea https://dynamiclibraryjdl.herokuapp.com/JDLCore . Esto cargará la función importarDesdeJDL que recibe como parámetros los identificadores de las funciones a importar.

Se recomienda utilizar la función importarDesdeJDL una vez que se tiene la seguridad de que la página se ha cargado totalmente. De lo contrario es posible que la función aún no se haya cargado y si se intenta utilizar se obtenga un error debido a que se intenta referenciar a una función que no existe.

NOTA: Si se requiere importar más de una función se debe llamar a la función de la siguiente manera importarDesdeJDL( idFuncion, idFuncion, idFuncion ).
